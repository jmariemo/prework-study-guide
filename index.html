HTML

  • HTML gives the content structure.
  • The head element contains information about the webpage, not visible to the user.
  • The body element represents the visible content shown to the user.
  • Element refers to the entire element from opening tag to closing tag, while tag refers only to what appears inside the angle brackets.

CSS

  • CSS gives the content style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

JavaScript

  • JavaScript gives the content function.
  • JavaScript Basics
  • JavaScript data types & data structures
  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • Interrupt control flow using conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
    Interrupting the control flow allows us to build apps that respond to a user's actions.
  • Condtional statement if:
    if (condition) {
    // block of code to be executed if condition is truthy
    }
    When computer encounters if statement in JS file, computer will check to see if the condition is truthy. If truthy, computer will do whatever action we ask between the curly brackets. If not truthy, computer will ignore instruction between curly brackets and move on.
  • Condtional statement multiple if:
    if (first condition) {
    // block of code to be executed if first condition is truthy
    } else if (second condition) {
    // block of code to be executed if second condition is truthy
    }
  • A variable is a named container that allows us to store data in our code.
